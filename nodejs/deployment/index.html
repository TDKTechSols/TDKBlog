Deployment

3.1 Prerequisites

  • - Delete 'node_modules' folder
  • - Create 'Procfile' file

web: node index.js

3.2 Heroku

  • Sign up an account from: https://signup.heroku.com
  • Menu New » Create new app:
    • App name: sonkk
    • Choose a region: Europe
  • Settings tab:
    • App name: sonkk
    • Heroku Git URL: https://git.heroku.com/sonkk.git
    • Domain: https://sonkk.herokuapp.com/

- Update 'utils/MyConstants.js' file

- Deploy tab » Deploy using Heroku Git:

  1. Login the Heroku CLI
  2. Create a new Git repository OR clone an existing Git repository
    • Create a new Git repository
    • Clone an existing Git repository
  3. Deploy your application
