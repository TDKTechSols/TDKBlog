Deployment

3.1 Prerequisites

  • Update config/app.php file
  • Update config/database.php file
  • Update config/mail.php file
  • Create ’Procfile’ file
    web: vendor/bin/heroku-php-apache2 public/

3.2 Heroku

Sign up an account from: Heroku

Menu New » Create new app:

  • App name: sonkk
  • Choose a region: Europe

Settings tab:

  • App name: sonkk
  • Heroku Git URL: https://git.heroku.com/sonkk.git
  • Domain: https://sonkk.herokuapp.com/

Deploy tab » Deploy using Heroku Git:

  1. Install the Heroku CLI
  2. Create a new Git repository OR clone the repository
    • Create a new Git repository
    • Clone the repository
  3. Deploy your application
