Preparation

1.1 Installation

1.1.1 PHP

  • Windows:
    • Download and unzip the zip file for lastest version from: https://windows.php.net/download
    • Rename $PHP_PATH$/php.ini-development file to php.ini
    • Add $PHP_PATH$ to PATH Environment variable
  • MacOS:
    • Download and install for lastest version from: https://php-osx.liip.ch
      $ curl -s http://php-osx.liip.ch/install.sh | bash -s 7.3
    • Export PATH
      $ export PATH=/usr/local/php5/bin:$PATH
  • To confirm PHP is installed on your machine: php --version

1.1.2 Visual Studio Code

  • Download and install from: https://code.visualstudio.com/download
  • PHP Server extension
    • Install PHP Server (brapifra) extension from:
      https://marketplace.visualstudio.com/items?itemName=brapifra.phpserver
    • Edit in settings.json:
      • "phpserver.phpPath": "$PHP_PATH$/php.exe" (for Windows)
      • "phpserver.phpPath": "/usr/local/php5/bin/php" (for MacOS)
      • "phpserver.port": 5050
    • test.php file: <?php phpinfo(INFO_MODULES); ?>
      » Right-click .php file » PHP Server: Reload Server

1.1.3 Composer

  • Windows: download and run Composer-Setup.exe from:https://getcomposer.org/doc/00-intro.md
  • MacOS: download and install for lastest version from:https://getcomposer.org
    • curl -s https://getcomposer.org/installer | php
    sudo mv composer.phar /usr/local/bin/composer
  • To confirm Composer is installed on your machine: composer -version

1.1.4 Create project

  • $ composer create-project laravel/laravel --prefer-dist shopping-online
  • Browser: http://domain/public/index.php
