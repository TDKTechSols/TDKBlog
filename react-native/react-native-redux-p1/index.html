Using Redux in your React Native App Part 1

In this tutorial, you will set up Redux in your React Native application. At the end of this tutorial, you will be able to:

  • Set up Redux for your app
  • Create reducers, actions and action creators

  • First, set up Redux (tested on v4.0.5) and allied packages by typing the following at the prompt in your project folder:
    • $ npm install redux react-redux redux-thunk redux-logger --save
